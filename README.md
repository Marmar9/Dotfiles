# My dotfiles
Focusing on preformance and productibility

Features: 
- Wofi app launcher
- Alacritty (for fast launches)
- Tmux for multiplexing
- Nvim for main editor 
- Themed with cattpuccin mocha

##  Installation
```bash
git clone
cd Dotfiles & (
./install
./setup
./sync )
```
Unfortunately not everything is automatic for now
- Select catpuccin mocha main and cursor theme in nwg-look software ( Will be automatic in future )
